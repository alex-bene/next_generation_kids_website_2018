Τίτλος : Ίδρυση της ομάδας
Περιεχόμενο : Η ομάδα NGK ιδρύεται τον Οκτώβρη του 2017 στο 2ο Δημοτικό Σχολείο Καλυβίων.
Χρονιά : Οκτώβριος 2017
Εικονίδιο : foundation

Τίτλος : Συμμετοχή σε Pre-Game event για το FLL
Περιεχόμενο : Η ομάδα συμμετέχει σε pre-game event για το  διαγωνισμό FLL με συμμετοχή πολλών  ομάδων της Αττικής. Εκεί για πρώτη φορά παρουσιάζει το project και τις αποστολές του διαγωνισμού στο Robot Game.
Χρονιά : Ιανουάριος 2018
Εικονίδιο : default

Τίτλος : Συμμετοχή στον προκριματικό διαγωνισμό FLL Αθήνας
Περιεχόμενο : Με τη συμμετοχή μας στον προκριματικό διαγωνισμό FLL της Αθήνας κερδίσαμε, εκτός απο την πρόκριση της ομάδας μας στον τελικό της Θεσσαλονίκης, την 3η θέση στη γενική κατάταξη ανάμεσα σε 37 ομάδες, την 3η θέση στο Robot Game καθώς και το 1ο Βραβείο Ομαδικότητας (Core Values)
Χρονιά : Φεβρουάριος 2018
Εικονίδιο : award

Τίτλος : Συμμετοχή στον Εθνικό Τελικό του FLL στη Θεσσαλονίκη
Περιεχόμενο : Η ομάδα μας, μετά την πρόκρισή της στον τελικό, προετοιμάστηκε σκληρά και έτσι κατάφερε να κερδίσει την 1η Πανελλαδική θέση με το βραβείο προγραμματισμού στο πεδίο του Robot Design, την 5η Πανελλαδική θέση στο Robot Game, την 2η καλύτερη βαθμολογία στο All Star Robot Game μεταξύ των 12 καλύτερων ομάδων της Ελλάδος καθώς και την 3η θέση στην τελική κατάταξη! Μάλιστα, αυτή η 3η θέση ήταν που της έδωσε το εισιτήριο για τον παγκόσμιο γύρο του FLL στην Εσθονία, <a href="https://www.firstlegoleague.ee/open2018/" target = "_blank">FIRST® LEGO® League Estonian Open International</a>
Χρονιά : Μάρτιος 2018
Εικονίδιο : award

Τίτλος : Εμφάνιση σε τηλεοπτική εκπομπή
Περιεχόμενο : Η ομάδα μας, μετά την πρόκρισή της στον παγκόσμιο γύρο του FLL στην Εσθονία, βρέθηκε προσκεκλημένη, μαζί με την ομάδα <a href="https://www.facebook.com/gyrosinnokids/" target = "_blank">Gyro's Innokids</a> στην πρεμιέρα της εκπομπής Ξυπνάμε Μαζί του Σπύρου Χαριτατου όπου και έγινε ένα αφιέρωμα στην ρομποτική 
Χρονιά : Μάρτιος 2018
Εικονίδιο : default

Τίτλος : Συμμετοχή στο Athens Science Festival
Περιεχόμενο : Η ομάδα μας συμμετήχε στο Athens Science Festival όπου και παρουσίασε το project και τις αποστολές του φετινού διαγωνισμού στους επισκέπτες. Το Athens Science Festival διοργανώνεται από τον εκπαιδευτικό οργανισμό «Επιστήμη Επικοινωνία – SciCo», το Βρετανικό Συμβούλιο, την Τεχνόπολη του Δήμου Αθηναίων, το Σύνδεσμο Υποτρόφων του Ιδρύματος Ωνάση και τη Γενική Γραμματεία Έρευνας και Τεχνολογίας σε συνεργασία με πολλούς ακαδημαϊκούς, ερευνητικούς φορείς και εκπαιδευτικούς οργανισμούς.
Χρονιά : Απρίλιος 2018
Εικονίδιο : default

Τίτλος : Συμμετοχή στο IT Festival στα Ιδιωτικά Εκπαιδευτήρια Νέα Γενιά Ζηρίδη
Περιεχόμενο : Σε αυτό το festival η ομάδα θα παρουσιάσει στο κοινό τι είναι ο διαγωνισμός FLL καθώς και το project της. Επίσης, θα παρουσιάζει στους επισκέπτες τις αποστολές του Robot Game. Το Φεστιβάλ είναι μια γιορτή της τεχνολογίας, ανοιχτή προς κάθε ενδιαφερόμενο γονέα, μαθητή, φοιτητή, ερευνητή στο πλαίσιο της οποίας μαθητές Δημοτικού, Γυμνασίου και Λυκείου προβάλλουν το ταλέντο και τη δημιουργικότητά τους στον ψηφιακό τομέα.
Χρονιά : Μάιος 2018
Εικονίδιο : default
